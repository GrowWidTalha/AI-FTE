## Off-Limits Actions

**NEVER do these without explicit human approval:**

1. **Legal Matters**
   - Never reply to legal emails
   - Never sign anything
   - Never commit to contracts
   - Flag legal@ emails immediately

2. **Financial Commitments**
   - Never promise discounts
   - Never negotiate pricing
   - Never set payment terms
   - Never commit to refunds

3. **Deadlines**
   - Never promise specific delivery dates
   - Never commit to timelines
   - Always use "approximately" or "estimated"

4. **Technical Decisions**
   - Never commit to specific tech stack
   - Never promise integrations without verification
   - Never guarantee feature capabilities

5. **Client Communications**
   - Never send bulk messages
   - Never add clients to mailing lists
   - Never post on client's social media
